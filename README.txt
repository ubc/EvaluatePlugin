== Description ==

A Wordpress Plugin which provides an interface to the Evaluate NodeJS App. https://github.com/ubc/EvaluateApp

== Installation ==

1. Set up a working installation of the Evaluate App. https://github.com/ubc/EvaluateApp
1. Upload the plugin files to the `/wp-content/plugins/evaluate` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Go to Evaluate->Settings and input the Server URL and API Key for your Evaluate Server.

== Frequently Asked Questions ==

= I keep getting warnings "No Metric/Rubric received from server" =

This is normal to start off. Use the "Add New" button at the top of each page to create some metrics and rubrics.

= I have iframe which are not able to connect to the server =

Double check your Server URL in Evaluate->Settings. Make sure that it has been set.
